## ?? UI COMPONENTS

### Top Navigation Bar
```
Furni. Admin | ?? Dashboard | ?? Products | ?? Orders | ?? Logout
```

### Buttons
```
?? Green  = Add/Save (Primary action)
?? Yellow = Edit (Modify action)
?? Red    = Delete (Danger action)
? Black  = Cancel/Back (Secondary action)
```

### Status Badges
```
?? Active   = Product visible in shop
? Inactive = Product hidden from shop
```

### Icons
```
? Plus    = Add new
??  Edit    = Modify
???  Trash   = Delete
?? Chart   = Dashboard
?? Boxes   = Products
?? Receipt = Orders
?? Door    = Logout
```
