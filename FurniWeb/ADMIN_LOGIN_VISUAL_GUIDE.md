## ? Testing Checklist

```
PRE-LOGIN
  ? App starts without errors
  ? No database errors
  ? /admin redirects to login

LOGIN TEST
  ? Can reach /Admin/Account/Login
  ? Login form displays correctly
  ? CSRF token present
  ? Email field required
  ? Password field required

VALID LOGIN
  ? admin@furni.com + Admin@12345
  ? Redirects to /Admin/Dashboard
  ? Navbar visible
  ? Dashboard shows metrics

INVALID LOGIN
  ? Wrong password ? Error shown
  ? Wrong email ? Error shown
  ? Empty fields ? Required error

NAVIGATION
  ? Dashboard link works
  ? Products link works
  ? Orders link works
  ? All pages load correctly

LOGOUT
  ? Logout button visible
  ? Click logout ? Signs out
  ? Redirects to home page
  ? /admin redirects to login again

AUTHORIZATION
  ? Non-admin user can't access /Admin/Products
  ? Shows Forbid/403 error
  ? Regular users can only see public pages
```
